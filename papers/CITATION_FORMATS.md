## Paper 1: Constraint Theory Foundation

### BibTeX

```bibtex
@article{constrainttheory2025,
  author = {SuperInstance Research Team},
  title = {Constraint Theory: A Geometric Foundation for Deterministic AI},
  journal = {arXiv preprint arXiv:2503.15847},
  year = {2025},
  primaryclass = {cs.LG}
}
```

### APA (7th Edition)

```
SuperInstance Research Team. (2025). Constraint theory: A geometric foundation 
for deterministic AI. arXiv preprint arXiv:2503.15847.
```

### MLA (9th Edition)

```

Deterministic AI." arXiv preprint arXiv:2503.15847 (2025).
```

### Chicago (17th Edition)

```

Deterministic AI." arXiv preprint, 2025. https://arxiv.org/abs/2503.15847.
```

### IEEE

```
Deterministic AI," arXiv preprint arXiv:2503.15847, 2025.
```

### Harvard

```

for Deterministic AI', arXiv preprint arXiv:2503.15847.
```

## Usage Notes

1. **Preprint Citation**: Before journal publication, cite using arXiv IDs
2. **Version Tracking**: Note the version number when citing
3. **Conference vs Journal**: Use appropriate format for venue
4. **Cross-References**: When citing multiple papers, use the complete series
