## Quick FAQ Reference

### "What is zero hallucination?"
```
P(constraint violation) = 0. Every output satisfies the manifold constraints.
Not a general AI solution - just geometric constraint guarantees.
Full proof in THEORETICAL_GUARANTEES.md
```

### "Is the benchmark fair?"
```
Compared against naive NumPy (brute force). Speedup from:
1. KD-tree O(log n) vs O(n)
2. Compiled Rust vs Python
3. Spatial indexing vs linear scan
Benchmark code in BASELINE_BENCHMARKS.md
```

### "Where would I use this?"
```
Potential use cases:
1. Numerical stability (robotics, graphics)
2. Deterministic ML (safety-critical)
3. Educational visualization
4. Research platform

Limitations: 2D only, specific constraints, early research
Looking for real-world use cases to test
```
