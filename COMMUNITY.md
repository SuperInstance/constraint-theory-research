## Getting Started

### New to Constraint Theory?

Welcome! Here's how to get started:

1. **Explore the Project:**
   - Read the [README](../README.md)
   - Check out the [documentation](https://constraint-theory.superinstance.ai)
   - Browse [tutorials](../tutorials/)
   - Explore [examples](../examples/)

2. **Join the Community:**
   - Star the repository on GitHub
   - Join our [Discussions](https://github.com/SuperInstance/constrainttheory/discussions)
   - Subscribe to updates

3. **Set Up Your Environment:**
   - Follow the [installation guide](../README.md#installation)
   - Run the tutorials
   - Try the examples

### First Steps

**For Developers:**
- Read [CONTRIBUTING.md](../CONTRIBUTING.md)

- Try the [quick start](../README.md#quick-start)

**For Students:**

## Getting Help

### Troubleshooting

**Before Asking:**
1. Check [TROUBLESHOOTING.md](../TROUBLESHOOTING.md)
2. Search existing [Issues](https://github.com/SuperInstance/constrainttheory/issues)
3. Search [Discussions](https://github.com/SuperInstance/constrainttheory/discussions)

**When Asking:**
- Provide context and error messages
- Share minimal reproduction code
- Include version information
- Be patient and respectful
- Explain mathematical context

**Where to Ask:**
- **Quick Questions:** GitHub Discussions
- **Bug Reports:** GitHub Issues
- **Math Questions:** GitHub Discussions with "research" tag
- **Security Issues:** security@superinstance.ai
- **Code of Conduct:** conduct@superinstance.ai

### Mentorship

**New Contributors:**
- Check [good first issues](https://github.com/SuperInstance/constrainttheory/issues?q=is%3Aissue+is%3Aopen+label%3A%22good+first+issue%22)
