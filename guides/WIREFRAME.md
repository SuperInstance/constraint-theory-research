**Version:** 1.0
**Date:** 2026-03-18
**Status:** Complete Design Specification

## Wireframe Overview

This document provides detailed wireframes for all pages in the constraint-theory website, following the mild cyberpunk aesthetic inspired by lucineer.

**Design Principles:**
- Dark theme with high-contrast accents
- Grid-based layouts for precision
- Progressive disclosure of complexity
- Interactive demonstrations
- Mobile-first responsive design
